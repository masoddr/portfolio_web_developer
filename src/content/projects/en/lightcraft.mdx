LightCraft is a portfolio website for a professional photography studio. The project aimed to showcase the studio's work in a visually stunning and interactive way.

Built with a focus on visual storytelling, the website features smooth animations and a gallery system that highlights the studio's photography work.

Key project features:
- Interactive photo galleries
- Client testimonials
- Booking system
- Blog integration
- Social media integration
